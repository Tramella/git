# Commands
<!-- Cài Đặt git vào dự án -->
- git init 

<!-- Xem tình trạng dự án  (tất cả các file hay folder) -->
- git status


<!-- Chuẩn bị lưu tại thời điểm hiện tại của dự án -->
- git add <tên-file>
   Ex: git add index.html
<!-- Tất cả các file -->
- git add . 

<!-- Hủy các chuẩn bị của git add -->
- git reset
